# Протокол обмена

ESP32-C3 отправляет в USB-Serial по одной JSON-строке на измерение (JSONL).

## Частота

- Период отправки: 500 мс
- Скорость порта: 115200 бод

## Формат сообщения

`{"deviceId":"esp32c3-01","sensorId":"light0","ts":1234567,"value":1872}`

## Поля

- `deviceId` (`string`) — идентификатор устройства
- `sensorId` (`string`) — идентификатор датчика
- `ts` (`number`) — время в миллисекундах от старта ESP32 (`millis()`)
- `value` (`number`) — сырое значение ADC, ожидаемый диапазон 0..4095

## Разделитель сообщений

- Каждое сообщение заканчивается переводом строки (`\n`)
